BeforeAfter

Simple JS for creating before after slider

View on GitHub Download .zip Download .tar.gz

What is this?

Let's face it. People love seeing how stuff looked before and after. That's why we wrote an easy to use vanilla javascript plugin for doing just that. No jquery required.

Cool, how do I use it?

Include the following files in your website
<script src="scripts/beforeafter.js"></script>
<link rel="stylesheet" href="css/BeforeAfter.css">

Markup:
<div id="SOME-ID">
<img class="img-responsive js-image-background" src="Background Image">
<img class="img-responsive js-image-mask" src="Mask Image">
</div>

Javascript:
var beforeAfter = new BeforeAfter({el:document.getElementById('SOME-ID'),options:{}});
